## 사용방법

#### 사용방법
<pre>
※ 일반회원

1. 회원가입
2. 로그인 (demo 페이지 기준 guest/guest0216 입력)
3. 게시판
 	3-1. 공지사항 게시판확인
	3-2. 일지 - 글작성
    3-3. 일지 - 조회
    3-4. 일지 - 상세보기
    3-5. 일지 - 삭제


※ 관리자

1. 게시판
    1-1. 공지사항 글 작성
    1-2. 공지사항 글 수정
    1-3. 공지사항 글 삭제
</pre>
